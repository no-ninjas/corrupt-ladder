Corrupt Ladder

An interactive fiction / gamebook / web game developed in 48h for Codebits 2012 competition.

Made by No Ninjas using Story Slicer gamebook engine developed by them. Code and styling by Dan Peddle. Code and ideas by PAngelo. Code and storyline by ps. Artwork by RosanaMC.

Soundtrack by Un and Nasienie with courtesy of Enough Records netlabel.

Tested under Chrome, Firefox and Safari.
Should work with Opera. Probably won't with Internet Explorer.
